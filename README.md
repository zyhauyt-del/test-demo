# 优化算法Demo

一些最优化方法的实现和测试，以前上课和自学时写的。

**文件说明：**

- `steepest_descent_newton.py` — 最速下降法 vs 牛顿法 Python 实现，含收敛路径对比图
- `constrained_nlp.m` — 约束非线性规划 MATLAB 实现，用 SQP 算法求解
- `最速下降法与牛顿法（Python实现）.docx` — 原始笔记（原理+代码）
- `约束非线性规划（MATLAB）.docx` — 原始笔记（原理+代码）

跑一下 `python steepest_descent_newton.py` 就能看到三种方法的对比结果。
